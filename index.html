How does JAVASCRIPT work ?

Ovbiously,javascript is a single threaded programming language.which is one of the most popular programming language.it works asynchronously and run in browser by V8 ENGINE.First of all, javascript code recieved by the "parser" inside the js engine.The parser find the syntax error line by line.If any error find out by parser,the parser show a error and sotp the execution to run code.If no error found in the code,then it creates the data structure call AST(Abstract Syntax Tree).After then,the engine convert the code into machine code.The converted code is sent to the system for excution.Finally the code run by the Engine.

That's how the javascript work.

What is the difference between LOCAL STORAGE and SESSION STORAGE ?

LOCAL STORAGE : Locat storage is not session based.If we need to delete any data form local storage, you need to use clear method of javascript as localStorage.clear() .That means, data will never expire in local storage. It is accessible for any window and it capacity 10mb. Local storage storing data locally.

SESSION STORAGE : Session Storage is based on session. When window or tab closed, then session storage will be closed. That means, data will be expired when colse tab or window. It is not accessible for any window, only for same tab and it capacity is 5mb. Session storage storing data for a while until tab close.

Local storage and session storage is similar. Local storage data not expired but session storage data will be expired. It is the main difference between local storage and session storage.
